Syntax Highlighting in the editor "notepad++"
-----------------------------------------

Jedit is a free editor, running on Windows.

Installation
- install notepad-plus
- copy the file "userDefineLang.xml"
  to the folder %appdata%\notepad-plus
  (It's a good idea to backup the file "userDefineLang.xml" if it already exists.)

Using
- Open your mavscript template in notepad++
- Menu "Language", choose "Mavscript"



Syntax-Hervorhebung im Editor "notepad++"
-------------------------------------

Jedit ist ein freier Texteditor. Er läuft auf Windows.

Installation
- notepad-plus installieren
- die Dateien "catalog" und "mavscript.xml"
  ins Verzeichis %appdata%\notepad-plus" kopieren
  (Es ist empfehlenswert eine Sicherheitskopie der Datei "userDefineLang.xml" anzulegen, falls eine solche bereits existiert.)

Gebrauch
- In notepad++ eine Mavscript-Vorlage öffnen
- Menü "Sprache", "Mavscript" wählen
