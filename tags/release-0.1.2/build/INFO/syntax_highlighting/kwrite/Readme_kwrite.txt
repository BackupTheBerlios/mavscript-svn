Syntax Highlighting in the editor "kwrite"
------------------------------------------

Kwrite is the KDE Text Editor, running on LINUX and UNIX platforms.

Installation
- copy the file "mavscript.xml"
  to the folder $HOME/.kde/share/apps/katepart/syntax

Using
- Open your mavscript template in kwrite
- Menu "Extras->Hervorhebungen->Sonstige", choose "Mavscript"
  in English it the menu item might be called "Extras->Highlighting->Others"
